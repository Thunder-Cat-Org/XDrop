[DOCUMENTATION](Docs.md)

To test, clone: `git clone https://github.com/Thunder-Cat-Org/token-drop`

### REQUIREMENTS

1. Python3.11 (exact version recommended)
2. Install pip

### Run

1. Create a python venv
   `python3.11 -m venv venv`

2. Activate
   `source venv/bin/activate`

3. Install xian contracting & pytest
   `pip install git+https://github.com/xian-network/xian-contracting pytest`
